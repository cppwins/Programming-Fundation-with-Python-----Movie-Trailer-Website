To run Movie Trailer Website demo,

On linux distro/ Mac OSX, install python 2.7.9, open a terminal emulater, 
go into directory where python program files are located, type 'python entertainment_center.py'

On windows system, install python 2.7.9, open IDLE(python GUI), open python
file 'entertainment.py', select Run --> Run module.
